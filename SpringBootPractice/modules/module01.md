## 1.2 SpringBootの起動
``課題２`` 下記の手順にしたがってSprigBootの起動とテストの実行を行ってください。

1. Dockerを起動する
```bash
cd docker
docker-compose up -d
```

2. SpringBootを起動する
```bash
cd ../
./gradlew bootRun
```

3. テストを実行する
```bash
./gradlew test
```
